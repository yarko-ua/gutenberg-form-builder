Advanced form builder for Gutenberg with custom fields, validation, and seamless integration.

== Description ==

Gutenberg Form Builder is a powerful WordPress plugin that extends Gutenberg with advanced form building capabilities. Create complex forms with custom fields, validation rules, conditional logic, and more.

**Features:**
* Drag & drop form builder
* Custom field types
* Conditional logic
* Form validation
* File uploads
* Email notifications
* PDF generation
* AJAX submissions
* Responsive design
* Translation ready

**Form Fields:**
* Text inputs
* Textareas
* Select dropdowns
* Radio buttons
* Checkboxes
* File uploads
* Date pickers
* And more...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gutenberg-form-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Start building forms using the Gutenberg editor

== Frequently Asked Questions ==

= How do I create a form? =

1. Create a new post or page
2. Add the "Form" block from the Gutenberg block inserter
3. Configure your form fields and settings
4. Publish your page

= Can I export form submissions? =

Yes, the plugin includes export functionality for form submissions and can generate PDF files.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes support for WPML.
